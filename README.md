# One-Shot Wonders

A website for One-Shot Wonders, an RPG gaming community based in Melbourne's northern suburbs.

## Tech stack

* NuxtJS
* DaisyUI
* Iconify icons
* Deployed with GitHub actions, NuxtHub and hosted on CloudFlare
